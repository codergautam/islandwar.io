# islandwar.io
A multiplayer game where you try to capture all the islands and dominate

## Setup & Running
### Debug
#### Building
```sh
npm install --include=dev
npm run build
```
#### Staring
Once the game is built, run `npm start` to execute it.
### Production
#### Building
```sh
npm install --include=dev
npm run prod
```
#### Starting
`npm start`
